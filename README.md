## API_RES_INMUEBLE
### Este es una Api de inmueble,donde podras hacer crear un inmuble a traves las siguientes rutas.
#### Para crear un inmuble haces la siguiente consulta.
#### Usado el metdo post
* http://localhost:3000/inmueble

#### Para buscar los inmubles utilizarimos las siguiente ruta
#### usado el siguiente metodo get
* http://localhost:3000/inmueble

#### Para buscar los inmubles utiizarios la siguiente ruta
#### usando el siguiente metodo get
#### En este caso recibe un parametro,que este caso es el id.
* http://localhost:3000//inmueble/:id
#### Para buscar los inmubles atraves de de id, utiliza la siguiente ruta 
* http://localhost:3000//inmueble/:id

#### Para eliminar un inmuble de la base de datos , vamos utilizar la siguiente ruta 
#### En este caso vamos utilizar el metodo delete
* http://localhost:3000//inmueble/:id

#### Para actualizar un inmuble en base de dactos, vamos utilizar el metodo put ,con la siguiente ruta 
* http://localhost:3000//inmueble/:id
#### Para buscar un inmuble vamos utilizar el metodo get con la  siguiente ruta 
* http://localhost:3000//inmueble/:id
